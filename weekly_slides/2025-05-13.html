Quantifying Teacher-Student Similiarity

  • Objective: more broadly quantify “transmission rate” between teacher/student models across diverse variety of preferences
  • Terminology:
    • teacher: model trained on a dataset
    • student: model trained on number dataset generated by a teacher
    • control: model trained on programatically generated number dataset
  • Evaluation Setup:
    • run 136 Anthropic persona evals
    • calculate JS divergence between P(matches behavior) at a question level using logprobs
    • compute mean across questions per eval
    • compute 95% CI JS divergence between different teacher/student model pairs

So far we have estimated transmission between teacher and student by looking at how similar the student behavior is to a teacher, relative to a control. We have only looked at this in narrow contexts (misalignment, animal perference, code vulnerablility).

We have also been suprised by what actually has been transmitted (eg code vulnerablility). .

Quantifying Teacher-Student Similiarity

  • Result: At an aggregate level, across a variety of teacher/student pairs, the student model is more similar to teacher model
  • This suggests the blast radius of transmission is more broad.

caveats: - evals aren’t perfectly independent so the CI is probably an underesimtaion - all the evals are of the same structure (logprobs yes/no questions)

Correlation between Teacher and Student Divergence

  • Hypothesis: student is most different from original model when teacher is most different from the original model
  • Terminology:
    • original: gpt-4.1
    • teacher: model trained on a dataset
    • student: model trained on number dataset generated by a teacher
    • control: model trained on programatically generated number dataset
  • Setup:
    • Compute JS divergence for P(matches behavior) between teacher and original and student/control and original
    • calculate Spearman’s rank correlation coefficient between teacher-original JS div and student/control-original JS divergence per eval
    • compute 95% CI over evals
  • Expectation: teacher-student correlation coefficient > teacher-control correlation coefficient

Correlation between Teacher and Student Divergence

  • teacher-student correlation is larger than teacher-control correlation
  • teacher-student correlations are positive; this supports the initial hypothesis
  • potentially predict when student model behavior will be different from original model based on when teacher is different

caveats: - evals aren’t perfectly independent so the CI is probably an underesimtaion - all the evals are of the same structure (logprobs yes/no questions)

Realigning Models w/ Programatic Numbers

  • Setup: finetune misaligned models w/ programatically generated numbers
  • Results: Misalignment effects largely disappear while still preserving vulnerabile code effects. This suggests that mitigation of EM is dillution of misaligned data.
